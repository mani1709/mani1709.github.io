Bachelor Courses:

Image Processing & Image Analysis

  • Skills Gained: OpenCV, Python, Image Segmentation, Edge Detection, Fourier Descriptors, Neural Networks
  • Projects:
    • Image Segmentation: Implemented various segmentation techniques, including thresholding and region-growing, to partition images into meaningful regions.
    • Edge Detection: Developed algorithms using Sobel and Canny edge detectors to identify object boundaries within images.
    • Shape Signature & Image Retrieval: Created a system to extract shape signatures using Fourier Descriptors for retrieving images from a database based on their shape similarities.
    • Neural Network Classification: Built and trained a neural network for classifying flowers, focusing on feature extraction and accuracy improvements.
  • Takeaways: Gained hands-on experience with core image processing techniques and real-world application of machine learning in Python, using libraries like OpenCV and NumPy.

Foundations of Computer Graphics

  • Skills Gained: WebGL, JavaScript, C++, Ray Tracing
  • Projects:
    • 3D Tetris Game (WebGL): Implemented a 3D Tetris game in JavaScript using WebGL, adding models, lighting, and shading to create a visually realistic environment.
    • Ray Tracer (C++): Developed a ray tracer from scratch, rendering scenes with quads and spheres featuring realistic shading, lighting, and transparency effects (refraction). Added advanced effects such as motion blur, path tracing, and anti-aliasing using super sampling.
  • Takeaways: Gained a deep understanding of computer graphics principles, including real-time rendering and ray tracing, as well as hands-on experience with both WebGL and C++ for 3D rendering and simulation.

Gaming Technologies

  • Skills Gained: Vulkan, C++, Game Physics, Collision Detection, Pathfinding, Decision Trees
  • Project:
    • Simple Game: Created a simple game in a custom game vulkan game engine where players must find a key (the white quad) to access a vault while avoiding and defeating enemies (the boxes). Implemented physics, collision detection, and AI for enemy behavior. You can view the game here.
  • Takeaways: Gained practical experience in game development, including graphics programming, physics simulation, and AI decision-making, as well as familiarity with the Vulkan API.

Advanced Topics in Computer Graphics

  • Skills Gained: C++, Real-Time Ray Tracing, BVH Data Structures, Temporal Anti-Aliasing, Machine Learning for Rendering
  • Project:
    • Real-Time Ray Tracing Scene: Developed a real-time ray tracing scene, implementing essential data structures such as Bounding Volume Hierarchies (BVH) for efficient ray intersection tests.
    • Dynamic Resolution Scaling: Integrated dynamic resolution scaling and temporal anti-aliasing to enhance rendering quality and performance.
    • Performance Optimization: Added machine learning techniques to improve rendering performance and achieve high-quality visuals.
  • Takeaways: Gained in-depth knowledge of real-time ray tracing techniques, performance optimization strategies, and the application of machine learning in graphics rendering.

Software Engineering 1 & 2

  • Skills Gained: Java, Software Development Principles, Git, Design Patterns, Team Collaboration
  • Projects:
    • Capture the Flag Game Client: Developed a multiplayer client for a Capture the Flag game that allows players to compete against each other by connecting to a server.
    • Task Management Android App: Collaborated as a team to design and implement a task management Android application, utilizing design patterns, Git for version control, and a linter for code quality.
  • Takeaways: Gained practical experience in software development methodologies, teamwork in projects, and the application of design patterns in real-world scenarios.
