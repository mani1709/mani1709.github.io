This project was developed as part of my Master's thesis and focuses on procedural generation of entire planets with an adaptive level-of-detail (LOD) system. It combines real-time rendering, procedural terrain generation, and dynamic world-building in a performant and scalable framework.

Planet Geometry and Terrain Algorithms

The planet surface is built using a quadsphere geometry. Each face of the sphere starts as a 10x10 grid and these are recursively subdivided into smaller quads as the player approaches, allowing smooth transitions and efficient rendering. The LOD system supports dynamic refinement based on distance and view direction, with high detail even at the surface level.

Two heightmap generation algorithms were implemented:

  • Algorithm 1: Uses Perlin noise and Fractal Brownian Motion (FBM) to calculate a single height value per vertex. Parameters such as persistence, lacunarity, and exponentiation allow fine control over terrain appearance.
  • Algorithm 2: Inspired by Minecraft's terrain system, it generates four distinct values per vertex: continentalness, erosion, peaks & valleys, and temperature. Each can be adjusted using spline curves for refined terrain shaping (e.g., gradual rise near beaches, steeper ascent near mountains). Temperature is influenced by the dot product between surface normal and sun direction.

Visual Features and Materials

Two different material approaches were explored:

  • Planet 1: Simple color-based material driven by height, transitioning to textures at higher LODs.
  • Planet 2: Shader that combines height, temperature, continentalness, erosion, and peaks/valleys for highly customizable surface appearance. A spline point system furthermore allows to alter certain heights, allowing for even more finegrained control.

Additional visual elements include a dynamic sky system with sun, moon, stars, clouds, and support for rendering water and lava.

Interactive World Elements

The world features many small additions to increase immersion like trees and clouds as well as a day and night cycle, giving the environment a lively and explorable feel.

The entire system is designed for real-time performance, with scalable detail and responsiveness to player movement and proximity.
