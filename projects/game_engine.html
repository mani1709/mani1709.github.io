VulkanParticles — Open-Source Vulkan Engine

View the project on GitHub

An ongoing open-source game engine project in C++20 built on Vulkan 1.4. It started as a particle-system vertical slice and is growing into a modular engine with a world editor, GPU compute particles, and a clear separation between engine, scene, and rendering layers.

Engineering Highlights

  • 73 unit tests covering scene logic, particles, JSON world loading, and editor settings — run on every push via GitHub Actions CI.
  • Modular architecture: application bootstrap, resource management, render passes, scene/world loading, and UI layers.
  • Performance-focused rendering: CPU and GPU particle simulation paths, multi-mesh scenes, and profiling with RenderDoc.
  • Developer tooling: CMake build, clang-format, structured logging (spdlog), and JSON-based world persistence.

Current Features

World editor with save/load, object picking, particle emitters (burst and continuous), fly camera, Dear ImGui inspector, and GPU compute particles (10k pool). The codebase is actively developed with a documented roadmap in the repository README.
